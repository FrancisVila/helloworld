Discovery Road Trip

Here's how we would do it if we were to learn this from scratch:

  1. If you don't have a login yet, click here to start your free trial.
  2. Follow the steps in Get started with App Builder to install the development environment on your system
  3. Follow the 2-part Hello World tutorial from Adelois Toto's blog, to develop your first app with the environment.
  4. Take a break watching a few videos from the Axway Developer YouTube channel, particularly Ahoy there Alloy! (32 minutes) and Dashboards or die! (14 minutes)
  5. Follow the Mobile Backend Services tutorial from the Axway Doc portal to discover how to pepper your apps with geolocalisation and push notifications.

Titanium

Installs on your system,
upload projects to cloud

What is the difference between Titanium, Appcelerator Studio, and AMPLIFY Application Builder, anyway?.

Titanium is the environment to develop applications that can run on Windows, Android, iOS. It is composed of Appcelerator Studio, the corresponding Command Line Interface, adapters for each OS, and other modules.

Appcelerator Studio is the IDE suggested by Axway to develop Titanium applications. You can also, if you prefer, develop using the Titanium extensions for VS Code or Atom, for example.

AMPLIFY Application Builder is a more general term referring to the Titanium environment, with added monitoring and control tools such as AMPLIFY Dashboard.
