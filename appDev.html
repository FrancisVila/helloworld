Studio is an Eclipse-based IDE that provides a single, extensible environment to rapidly build, test, package, and publish mobile apps across multiple devices and OSs.

CLI

the Command Line Interface for Titanium

What is CLI?

CLI is a set of command-line tools to develop and deploy mobile and cloud applications. CLI provides a unified interface for using the other command-line tools, such as the Titanium CLI, Alloy CLI and API Runtime Services CLI.

Hyperloop

Reach even the most obscure mobile OS-specific capabilities via native APIs, directly from JavaScript when using Titanum.

What is Hyperloop?

Hyperloop gives you direct access to the native mobile APIs. Titanium SDK already covers plenty of the native APIs and makes them cross-platform addressing the most popular APIs. This technology was built for those developers who require access to even the most obscure APIs out there.

What is Dashboard?

An integrated web application with user interface to manage all aspects of your app development. It allows you to manage all your mobile applications in your organization, native cross-platform Titanium applications, as well as iOS, Android, Windows and Node.js applications.

Discovery Roadtrip

Here's how we would do it if we were to learn to use Axway application development from scratch:

  1. If you don't have a login yet, click here to start your free trial.
  2. Follow the steps in Get started with App Builder to install the development environment on your system
  3. Follow the 2-part Hello World tutorial from Adelois Toto's blog, to develop your first app with the environment.
  4. Take a break watching a few videos from the Axway Developer YouTube channel, particularly Ahoy there Alloy! (32 minutes) and Dashboards or die! (14 minutes)
  5. Follow the Mobile Backend Services tutorial from the Axway Doc portal to discover how to pepper your apps with geolocalisation and push notifications.
